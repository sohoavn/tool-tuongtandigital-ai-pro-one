❓ Trung tâm trợ giúp

🚀 Bắt đầu trong 5 phút

1
Lấy API Key miễn phí

Truy cập Google AI Studio, đăng nhập bằng tài khoản Google, nhấn "Create API Key". Sao chép key.

2
Nhập API Key vào ứng dụng

Nhấn biểu tượng 🔑 góc trên phải → Dán API Key → Nhấn "Thêm Key". Ứng dụng tự kiểm tra key hợp lệ.

3
Upload tài liệu

Trong tab Notebook → nhấn "Tải file" → chọn PDF, Word, ảnh, audio, hoặc dán URL trang web.

4
Bắt đầu hỏi AI

Gõ câu hỏi vào ô chat → nhấn Enter hoặc Ctrl+Enter. AI sẽ trả lời dựa trên nội dung tài liệu của bạn.

5
Khám phá tính năng nâng cao

Thử tạo Podcast AI 🎙️, Flashcard 🃏, Quiz 📊, Mind Map 🗺, hoặc chuyển đổi file với tab Convert.

📚 Notebook AI

Upload tài liệu hiệu quả

Hỗ trợ: PDF, TXT, MD, DOCX, ảnh (PNG/JPG/WEBP), audio (MP3/WAV/M4A), video (MP4). Tối đa 20MB mỗi file.

Mẹo: Upload nhiều tài liệu liên quan để AI tổng hợp thông tin xuyên suốt. Dùng tab "YouTube" để trích xuất transcript video tự động.

Chat AI hiệu quả

  • Đặt câu hỏi cụ thể: "Tóm tắt ý chính của chương 3" thay vì "Tóm tắt"
  • Yêu cầu so sánh: "So sánh quan điểm của tác giả X và Y về vấn đề Z"
  • Đề nghị ví dụ: "Cho 3 ví dụ thực tế về khái niệm này"
  • Dùng Persona AI: chọn 🎓 Giáo sư để phân tích học thuật, 💼 Chuyên gia để phân tích kinh doanh

Multi-Notebook Manager

Tạo tối đa 20 notebooks riêng biệt. Mỗi notebook lưu riêng tài liệu và lịch sử chat. Nhấn biểu tượng ☰ để mở sidebar quản lý. Double-click tên để đổi tên nhanh.

Tạo Podcast AI 🎙️

Nhấn nút "🎙️ Podcast" trong toolbar. AI sẽ: (1) Viết kịch bản đối thoại 2 nhân vật, (2) Dùng Gemini TTS đọc từng lượt thoại với 2 giọng khác nhau, (3) Ghép thành 1 file WAV để tải về.

Lưu ý: Tính năng này cần nhiều API calls. Nên dùng khi có nhiều API keys để tránh rate limit.

🎤 Speech to Text

Định dạng file tốt nhất

Ưu tiên: WAV, MP3 chất lượng cao, M4A. Tránh file có tiếng ồn nền cao. Kích thước tối ưu: dưới 10MB.

Chế độ phiên âm

  • Chuẩn: Phiên âm đơn giản, nhanh nhất
  • ⏱ Timestamp: Thêm dấu thời gian [00:01:23] vào từng đoạn
  • 👥 Phân biệt người nói: Tự động nhận diện và gán nhãn Người nói 1, 2... (phù hợp cuộc họp, phỏng vấn)

Mẹo chất lượng cao

Bật "Phân tích nội dung sau phiên âm" để AI tự động tóm tắt, trích điểm chính và gợi ý action items ngay sau khi phiên âm xong.

Reading Display Panel

Khi TTS đang phát, panel "Đang đọc" sẽ hiển thị văn bản và highlight từng từ/câu đang được đọc theo thời gian thực. Nhấn A+/A- để điều chỉnh cỡ chữ.

⏺️ Ghi hình & Biên bản

Cách tạo biên bản cuộc họp tự động (3 bước)

1
Chọn chế độ ghi – Màn hình (kèm tiếng), Camera, hoặc Chỉ âm thanh. Chọn template biên bản phù hợp.
2
Nhấn "Bắt đầu ghi" – Trình duyệt yêu cầu quyền truy cập. Ghi hình cuộc họp của bạn.
3
Nhấn "Dừng & Xử lý" – AI tự động phiên âm và tạo biên bản, action items, tóm tắt. Xuất DOCX hoặc thêm vào Notebook.

Lưu ý: File >50MB sẽ được cắt đoạn tự động. Kết nối internet ổn định giúp xử lý nhanh hơn.

⚡ Convert & Phân tích

Các hành động AI

  • 📝 Trích xuất: Lấy toàn bộ nội dung văn bản từ file
  • 🌐 Dịch thuật: Dịch sang 8 ngôn ngữ, giữ nguyên cấu trúc
  • 📊 Tóm tắt: Tóm tắt ngắn gọn với các điểm chính
  • 🔍 Phân tích: Phân tích sâu nội dung, luận điểm, cấu trúc
  • ✨ Viết lại: Rewrite rõ ràng, cải thiện văn phong
  • 🗺 Mind Map: Tạo sơ đồ tư duy tương tác từ nội dung
  • ⚡ Tuỳ chỉnh: Nhập bất kỳ yêu cầu AI nào

Batch Mode

Xử lý nhiều file cùng lúc. Chọn tab "📦 Batch" → upload nhiều file → chọn hành động AI → nhấn Xử lý. Kết quả hiển thị riêng từng file, có thể tải về tất cả dạng ZIP.

✨ Tính năng AI đặc biệt

🃏 Flashcard (Alt+F)

Tạo 10–50 thẻ ghi nhớ từ tài liệu. Dùng bộ lọc "Chưa thuộc / Đã thuộc / Cần ôn" để ôn tập hiệu quả. Xuất CSV để dùng với Anki hoặc Google Sheets.

📊 Quiz (Alt+Q)

Tạo 5–50 câu trắc nghiệm 4 lựa chọn. AI tự chấm điểm và giải thích đáp án đúng. Xuất ra PDF để in.

🗺 Mind Map

Trong tab Convert, chọn hành động "🗺 Mind Map". AI tạo sơ đồ SVG tương tác. Zoom bằng cuộn chuột, kéo để pan, click node để mở rộng/thu gọn. Xuất PNG.

⚖️ So sánh tài liệu

Upload 2+ tài liệu → nhấn ⚖️ trong chat toolbar → chọn nguồn cần so sánh. AI tạo bảng ma trận so sánh theo tiêu chí. Xuất Markdown hoặc HTML.

📖 Từ điển thuật ngữ

Nhấn 📖 trong chat toolbar. AI tự động trích xuất và định nghĩa các thuật ngữ chuyên ngành trong tài liệu. Tìm kiếm A–Z, xuất TXT/DOCX.

🎙️ Gemini Live Voice Chat

Nhấn biểu tượng microphone trong chat input để kết nối Gemini Live API. Nói câu hỏi trực tiếp – AI sẽ phân tích tài liệu và trả lời bằng giọng nói tức thì. Fallback về Web Speech API nếu Live API không khả dụng.

⚙️ Cài đặt

Quản lý Model AI (T-M)

Ứng dụng tự động fetch danh sách model Gemini mới nhất mỗi 24 giờ. Khi có model mới, badge "✦ Mới" sẽ xuất hiện trên tab Settings. Bạn có thể chọn:

  • ⚡ Cân bằng (mặc định): Flash cho chat thông thường, Pro cho tác vụ nặng
  • 🚀 Nhanh & Miễn phí: Dùng Flash Lite cho tất cả
  • 💎 Chất lượng cao: Dùng Pro cho tất cả tác vụ

Multiple API Keys

Thêm nhiều Gemini API Keys để ứng dụng xoay vòng tự động, tránh lỗi rate limit. Mỗi key có giới hạn 15 RPM và 1500 RPD ở tier miễn phí.

❓ Câu hỏi thường gặp

Ứng dụng có lưu dữ liệu lên server không?

Không. Toàn bộ dữ liệu (tài liệu, chat history, cài đặt) được lưu cục bộ trên thiết bị của bạn qua IndexedDB. Chỉ nội dung file và câu hỏi của bạn được gửi đến Gemini API để xử lý.

API Key có bị lộ không?

API Key được lưu trong localStorage/IndexedDB của trình duyệt, không gửi đi đâu ngoài các request đến Gemini API. Không chia sẻ máy tính/trình duyệt với người khác nếu lo ngại bảo mật.

Lỗi "API key not valid" là sao?

Key đã nhập sai hoặc hết hạn. Vào AI Studio tạo key mới. Đảm bảo không có khoảng trắng thừa khi dán.

Lỗi 429 Too Many Requests là sao?

Đã vượt rate limit miễn phí (15 req/phút hoặc 1500 req/ngày). Giải pháp: thêm nhiều API keys, đợi 1 phút, hoặc nâng cấp lên Gemini API trả phí.

File PDF không đọc được nội dung?

PDF dạng scan (ảnh) cần bật OCR. PDF được mã hóa/password-protected không hỗ trợ. File >20MB quá lớn, cần tách ra.

Podcast AI mất bao lâu?

Tùy độ dài script (800–3000 từ) và số lượt thoại. Thường 2–5 phút. Đảm bảo không đóng tab trong quá trình tạo.

Có thể dùng offline không?

Giao diện có thể tải offline sau lần đầu (PWA). Nhưng tính năng AI đều cần kết nối internet để gọi Gemini API.

Dữ liệu có mất khi xóa cache trình duyệt không?

Có. Dữ liệu lưu trong IndexedDB sẽ mất nếu xóa dữ liệu trang web. Nhớ xuất dữ liệu (Settings → Xuất dữ liệu JSON) để backup định kỳ.

Hỗ trợ ngôn ngữ nào?

Gemini API hỗ trợ 100+ ngôn ngữ. Tốt nhất với Tiếng Việt, Tiếng Anh, Trung, Nhật, Hàn, Pháp, Đức, Tây Ban Nha.

Model Gemini nào đang được dùng?

Ứng dụng tự động fetch và chọn model tối ưu. Xem chi tiết tại Settings → Model AI. Mặc định dùng Gemini 2.5 Flash cho tác vụ thông thường và 2.5 Pro cho tác vụ nặng.

Tính năng "So sánh tài liệu" hoạt động thế nào?

Upload 2+ tài liệu vào Notebook → nhấn ⚖️ → chọn tài liệu cần so sánh. AI phân tích và tạo bảng ma trận so sánh theo nhiều tiêu chí như nội dung, luận điểm, phong cách, v.v.

Làm thế nào để cài đặt như app (PWA)?

Trên Chrome/Edge: nhấn biểu tượng cài đặt (⊕) trên thanh địa chỉ hoặc chờ banner "Cài đặt TuongTanDigital-AI" xuất hiện. Trên iOS Safari: nhấn Share → Add to Home Screen.

💡 Tips nâng cao

Tối ưu rate limit miễn phí

Thêm 3–5 API Keys khác nhau trong Settings. Ứng dụng tự động xoay vòng keys, giúp bạn thực hiện nhiều requests hơn trước khi bị giới hạn.

Prompting hiệu quả

Dùng Prompt Library (📚 trong chat toolbar) với 30+ template chuyên ngành. Chọn đúng Persona AI (🎭) phù hợp với loại tài liệu để có kết quả tốt hơn.

Workflow nghiên cứu

  1. Upload nhiều bài báo/tài liệu liên quan vào 1 notebook
  2. Dùng "So sánh tài liệu ⚖️" để xem tổng quan
  3. Chat để đào sâu từng điểm thú vị
  4. Ghim những câu trả lời quan trọng vào Pinned Notes
  5. Tạo Flashcard từ nội dung để ôn tập
  6. Xuất notebook thành DOCX để viết báo cáo

Backup dữ liệu định kỳ

Vào Settings → "Xuất dữ liệu (JSON)" để backup toàn bộ notebooks, sources và lịch sử chat. Dùng "Nhập dữ liệu" để restore khi cần.
